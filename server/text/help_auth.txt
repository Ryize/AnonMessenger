Для регистрации отправьте запрос на register/
Параметры: login: str (имя пользователя).
return: str (код для входа)

Для удаления аккаунта отправьте запрос на delete/
Параметры: code: str (код полученный при регистрации).
return: bool (True - аккаунт удалён, False - нет)

Для взаимодействия с другими клиентами, отправляйте в каждом запросе параметр code, который вы получили при регистрации
